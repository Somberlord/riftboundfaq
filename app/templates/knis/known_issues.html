Riot Rules

Some cards of effects might not yet be clear in the CRD yet, but Riot keeps interacting with the community and answered some of our questions.

Cithria of Cloudfield & Wildclaw Shaman (Buffs as Game Objects)

If Cithria of Cloudfield has no buffs and Wildclaw Shaman is played, both triggers are simultaneous and the player can order them as they choose. Cithria of Cloudfield can first receive a buff, then Wildclaw Shaman can spend it to receive a buff and ready itself.
Discord Message

The ability for buffs to be targeted is expected to be removed. Therefore, Wildclaw Shaman can spend any buff you control when it resolves.
Discord Message

Fox-Fire Targets Becoming Illegal

If Fox Fire targets become illegal (through might increase), Fox Fire's controller can choose which targets to 'deselect' until the Might of the remaining targets equals or drops below 4
Discord Message

Playing Hidden Non-Unit Cards in Response to Conquer Triggers

You cannot play a hidden card you control on a battlefield you no longer control.
Discord Message

Sett Legend Ability

Rule 572.2. is incorrect and is due to be removed. Sett's Legend ability should only be a replacement effect, not a Triggered Ability. Replace the card text's "when" with "if".
This intercedes on any other process occurring.
Discord Message

Dune Drake

Errata : the +2 should only last until end of turn.
Discord Message

Portal Rescue

Errata : The unit is played by it's owner to its base.
Discord Message

Dreaming Tree

Dreaming tree only triggers on things that target.
Discord Message

Baited Hook

Baited Hook uses the current might of the killed unit, but ignores only printed cost of the played unit.
Discord Message

Legion abilities

An ability with the legion keyword can't be activated unless another card was played.
Discord Message

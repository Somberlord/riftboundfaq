### Déplacement (Move)

Le **déplacement** désigne le mouvement d’un **Objet de jeu** entre deux **Emplacements** (**Battlefields** et **Bases**).

  - Actuellement, seules les **Unités** peuvent se déplacer.
  - Les objets de jeu peuvent être déplacés entre différents **Battlefields**, ou entre un **Battlefield** et la **Base** du joueur qui le contrôle.
  - Les **Unités** ne peuvent pas être déplacées vers un **Battlefield** déjà occupé par des **Unités** de deux autres joueurs.

Les effets qui permettent un déplacement peuvent spécifier des restrictions supplémentaires.

## Déplacement standard

Toutes les unités ont accès au Déplacement standard :

  - Ne peut être effectué que pendant son propre tour, en dehors d’un **Showdown** et avec une **Chaîne** vide.
  - Plusieurs **Unités** peuvent utiliser le **Déplacement standard** en même temps.
  - Le coût pour chaque **Unité** pour se déplacer ainsi est de s’**Exhaust**.
  - Les **Unités** se déplaçant ensemble via un **Déplacement standard** doivent avoir la même **destination**.
  - Les **Unités** se déplaçant ensemble peuvent provenir d’emplacements différents.
  - Les **Unités** peuvent utiliser le **Déplacement standard** pour aller de la **Base** vers un **Battlefield**, ou l’inverse.
  - Les **Unités** avec **Ganking** peuvent utiliser le **Déplacement standard** pour aller d’un **Battlefield** à un autre.
  - Plusieurs **Unités** ayant **Ganking** peuvent quand même se déplacer simultanément avec un **Déplacement standard**.

Toutes les autres restrictions de déplacement restent applicables.

## Notes

  - Seul le **Déplacement standard** épuise les unités. Tous les autres types de déplacement n’épuisent pas les unités, et peuvent déplacer des unités déjà épuisées.
  - Le **Déplacement standard** NE PEUT PAS être combiné avec d'autres effets de déplacement. (ex : capacité de légende de Yasuo)
